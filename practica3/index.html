  • <article>

    • Etiqueta article Un atículo debe tener sentido por sí mismo
    y debe ser posible distribuirlo independientemente del resto del sitio.

    Publicación en el foro
    Entrada en el blog

    Sin embargo,puede usar CSS para darle diseño.

  • <aside>

    • Etiqueta asideDefine algún contenido
    además del contenido en el que se coloca, el contenido aparte debe estar indirectamente relacionado con el contenido circulante
    Sugerencia:Aside se coloca a menudo como una barra lateral en un documento.
    Nota:Este elemento no se muestra como algo especial en el navegador.Sin embargo puede usar CSS para diseñar el aside

  • img

    • Etiqueta imagen Se utiliza para incrustar una imagen en una página HTML
    Técnicamente las imágenes no se insertan en una página web, las imágenes están vínculadas a páginas web.
    La etiqueta <ima> crea un espacio de espera para la imagen de referencia. Tiene atributos obligatorios:

  • src: Especifica la ruta a la imagen
  • alt: Especifica un texto alternativo para la imagen si por algún motivo no se puede mostrar

  • map

    • Etiqueta map Se utiliza para definir un mapa de imagen
    es una imagen con áreas en las que se puede hacer click.El atributo de nombre requerido está asociado
    El atributo de nombre requerido del <map>está asociado con el atributo usemap de <img>
    y crea una relación entre la imagen y el mapa. Ejemplo

  • script

    • Etiqueta script Se utiliza para incrustar un script del ladodel cliente JavaScript
    Contiene declaraciones de secuencias de comandos o apunta a un archivo de secuencia de comandos externos
    a través del atributo src.
    Los usos más comunes de JavaScript son manipulación de imágenes, la validación de formularios y los cambios dinámicos de contenido.
    Nota:Cuando un navegador lee una hoja de estilo formateará el documento HTML
    de acuerdo con la información de la hoja de estilo.Si se han definido algunas propiedades para el mismo slector
    (elemento) en diferentes hojas de estilo, se utilizará el valor de la última hoja de estilo leída(consulte el ejemplo a continuación.)
