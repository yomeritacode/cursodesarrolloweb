Tipos de etiquetas

  • <a>

    • Etiqueta ancla Sirve para colocar
    un URL de alguna pagina a un documento

    Soy un link

  • <abbr>

    • Etiqueta abbr Sirve para asignar
    una abreviatura

    HTML

  • <address>

    • Etiqueta address Define la información de contacto del autor/propietario
    de un documento u artículo

    Escrito por Alfonso6z

  • <article>

    • Etiqueta article Un atículo debe tener sentido por sí mismo
    y debe ser posible distribuirlo independientemente del resto del sitio.
    Fuentes potenciales para el elemento article:
    Publicación en el foro
    Entrada en el blog
    Noticia
    Nota:<article>no se muestra como algo especial en el navegador
    Sin embargo,puede usar CSS para darle diseño.

    Ejemplo article con css

  • <aside>

    • Etiqueta asideDefine algún contenido
    además del contenido en el que se coloca, el contenido aparte debe estar indirectamente relacionado con el contenido circulante
    Sugerencia:Aside se coloca a menudo como una barra lateral en un documento.
    Nota:Este elemento no se muestra como algo especial en el navegador.Sin embargo puede usar CSS para diseñar el aside

    Ejemplo aside con CSS

  • <form>

    • Etiqueta formEs un contenedor para diferentes tipos de elementos de entrada
    como :campos de texto, casillas de verificación, botones de opción, botones de envío,etc.

    Muestra un campo de entrada de texto de una sola linea

    Muestra un botón de radio para seleccionar una de muchas opciones)

    Muestra una casilla de verificación (para seleccionar cero o más opcioness)

    Muestra un botón de envío (para enviar el formulario)

    Muestra un botón en el que se puede hacer click

  • <footer>

    • Etiqueta footerDefine un pie de página para un documento o sección

  • Normalmente contiene:
  • Información de autoria
  • Información de registro
  • Información de contacto
  • Mapa del sitio
  • Volver a los enlaces superiores
  • Documentos relacionados

    • Ejemplo footer

  • <header>

    • Etiqueta headerRepresenta un contenedor de contenido introductorio
    o un conjunto de enlaces de navegación. Normalmente contiene:

  • Uno o más elementos de encabezado (<h1>-<h6>)
  • Logotipo o icono
  • Información de autoría
    • Nota:Puede tener varios <header>en un documento HTML.Sin embargo,
    no se puede colocar dentro de un <footer>,<address> u otro <header>
